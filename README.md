# Omnichannel AI Chatbot 🤖

An advanced AI-driven chatbot system integrated with **WhatsApp, Instagram, Telegram, and Facebook Messenger**.

### 🚀 Key Features
- **Omnichannel Support:** Single backend for multiple messaging platforms.
- **RAG Integration:** Uses Retrieval-Augmented Generation to provide accurate answers from custom documents.
- **Voice-to-Text:** Integrated with Whisper API to handle voice notes.
- **Flask Backend:** Fast and scalable Python-based server.

### 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Flask, LangChain
- **APIs:** Meta Graph API, Telegram Bot API, GROQ API
- **Database:** Qdrant / SQLite
